- Displays a signed-off timestamp for signed conversations.
- Handles conversations without an application.
- Creates a new internal application.
- Generates a PRD document from conversation messages.
- Generates a feature request document from conversation messages.
- Creates a stub research document for the conversation.
- Includes the application name in feature request documents.
- Notifies admin users when a document is created (with a link to the conversation).
- Notifies admin users when a research document is created.
- Dispatches a PRD generation job when signing off a new application conversation.
- Dispatches a PRD generation job when signing off a feature request conversation.
- Does not dispatch PRD generation for feature requests when signing off a new application conversation.
- Does not dispatch PRD generation for new applications when signing off a feature request conversation.
- Dispatches a research alternatives job when signing off a new application conversation.
- Does not dispatch the research alternatives job when signing off a feature request conversation.
- Dispatches a job when an automated application is created.
- Does not dispatch a job when a non-automated application is created.
- Dispatches a job when an application is updated to automated.
- Does not dispatch a job when an automated application is updated but is_automated unchanged.
- Does not dispatch a job when an application is updated from automated to non-automated.
